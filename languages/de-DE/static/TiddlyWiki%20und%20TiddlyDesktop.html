TiddlyWiki und TiddlyDesktop

ChrisK vor 5 Tagen

TiddlyDesktop ist eine Programm für Windows, Mac OS X und Linux, mit dem Sie TiddlyWiki Dateien bearbeiten können.

  1. Installieren Sie die aktuelle TiddlyDesktop Version von: https://github.com/Jermolene/TiddlyDesktop/releases
  2. Starten Sie TiddlyDesktop
  3. Verwenden Sie den "browse button" um TiddlyWiki Dateien zu öffnen.
  4. Speichern der Änderungen funktioniert wie gewohnt.

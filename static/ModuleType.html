ModuleType

9 months ago

The module-type field of a JavaScript module is a string that identifies the type of the module.

The module types used in this wiki are listed below.

  • allfilteroperator
  • animation
  • command
  • config
  • filteroperator
  • global
  • info
  • isfilteroperator
  • library
  • macro
  • parser
  • saver
  • startup
  • storyview
  • tiddlerdeserializer
  • tiddlerfield
  • tiddlermethod
  • upgrader
  • utils
  • utils-node
  • widget
  • wikimethod
  • wikirule
